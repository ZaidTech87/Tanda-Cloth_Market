## 💡 Important Notes

### Database
- Default database name: `cloth_marketplace`
- Update credentials in `application.properties`
- Schema auto-creates tables via JPA

### File Uploads
- Stored in `backend/uploads/` directory
- Max size: 50MB per file
- Supported: Images (JPG, PNG) and Videos (MP4, WEBM)

### Security
- JWT tokens expire in 24 hours
- Passwords encrypted with BCrypt
- CORS configured for localhost

### Chat System
- Uses polling (3-second interval)
- Voice messages in WebM format
- Chat history persists in database
